Pipeline

  1. 1
    Parse intent

    Extract style, site, scale, materials, constraints, and design brief.

  2. 2
    Plan space graph

    Generate rooms, privacy levels, edges, circulation, and floor hints.

  3. 3
    Retrieve templates

    Use KB v2 case cards, laws, and reviewed learning areas as references.

  4. 4
    Develop concepts

    Generate, rank, select, or fuse explainable design concepts before construction.

  5. 5
    Synthesize geometry

    Turn semantic plans into shells, rooms, roofs, doors, stairs, interiors, and site modules.

  6. 6
    Export datapack

    Write Minecraft Java 1.21 commands, preview, report, scorecard, concept, and critic artifacts.

  7. 7
    Evaluate and critique

    Run QA, repair hints, score rubric, command optimization, benchmark reporting, and Critic Council review.

Current Capabilities

  • Natural language to buildable datapacksMock mode works without API keys; LLM mode can supply semantic JSON.
  • Deterministic geometry engineCSG, BSP, A*, structure modules, facade, roof, site, and interiors.
  • Template Knowledge Base v2Reviewed case cards, retrieval index, design laws, and explainable references.
  • Concept StudioGenerates, ranks, selects, or fuses multiple explainable concepts before construction.
  • Critic CouncilBuildability, connectivity, habitation, style, composition, and site critics produce repair guidance; use `--no-critics` to disable it.
  • Benchmark suiteFixed Architecture Master prompt set, summaries, gallery, CSV, and feedback template.
  • Repair readinessStage 1 closed the visible site and comfort repair queue.
  • Extensible architectureAgents and evaluators stay modular around `src/construction`.

Roadmap

  1. Stage 1 - Baseline ReadinessCompleted

    Stable benchmark floor, scorecard, and repair closure evidence.

  2. Stage 2 - Template KB v2Completed

    Reviewed knowledge units, retrieval explanations, and design laws.

  3. Stage 3 - Concept StudioCompleted

    Generate, compare, select, or fuse explainable design concepts per prompt.

  4. Stage 4 - Critic CouncilCompleted

    Separate critics summarize buildability, habitation, style, composition, connectivity, and site repair guidance.

  5. Stage 5 - Neural RetrievalIn progress

    Artifacts-first neural assist suggests labels, builds deterministic embedding indexes, and evaluates fusion retrieval while default generation remains rule-only.
